HTML

  • Always close your tags
  • Style using sheets!
  • Use a validator
  • Comment!
  • Use relative URLs for images if possible or use URLs relative to the document root

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Commit, Pull, Push
  • Commit - Record changes to the repository
  • Pull - Fetch from and integrate with another repository or a local branch
  • Push - Update remote references along with associated objects

JavaScript

  • Variables defined with var are dynamic and can hold any variable type
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
